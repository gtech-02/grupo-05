# Digital Store

Bem-vindo ao **Digital Store**, um projeto desenvolvido pelo **Grupo 5** como parte das atividades práticas das aulas do curso de Full Stack do Geração Tech da IEL Ceará. Este e-commerce foi concebido com o objetivo de aplicar e consolidar os conhecimentos adquiridos durante as aulas, proporcionando uma experiência prática no desenvolvimento Front-End.
